Start with the runtime boundary

Get started

Miru stays small by delegating windows, input, rendering, fonts, and layout primitives to Soluna.

01

Add Miru beside Soluna

git submodule update --init --recursive
cp miru.lua your-app/miru.lua
02

Create a view

local miru = require "miru"

local view = miru.new {
    width = args.width,
    height = args.height,
    component_path = "components/?.lua;?.lua",
}
03

Mount stateful components

local count = miru.value(0)

return function()
    miru.vbox({ width = 320, gap = 12 }, function()
        miru.text("Count: " .. count())
        miru.mount("button", {
            label = "Increment",
            on_click = function()
                count(count() + 1)
            end,
        })
    end)
end

One predictable loop

Mental model

  1. ValuesRead reactive state during component render.
  2. LayoutCompose box, hbox, vbox, canvas, and text nodes.
  3. InteractionAttach click, focus, scroll, and dismiss behavior.
  4. FrameUpdate animations, then draw one Soluna batch.

Small public surface

Core API

The runtime separates view ownership, reactive primitives, layout nodes, and interaction modifiers.

miru.newOwn the root tree, input routing, animation clock, layout, and draw lifecycle.
miru.valueCreate dependency-tracked mutable state with callable read/write syntax.
miru.computedDerive cached values from other reactive dependencies.
miru.animatedMove a numeric target through a duration and easing function.
miru.mountResolve a component module and preserve its local reactive scope.
box / hbox / vboxBuild Yoga-backed layout nodes with absolute, flex, and overflow behavior.
miru.canvasWrite Soluna command streams inside a measured layout rectangle.
miru.textRender measured styled text with inherited text registries.
clickable / focusableRoute pointer, keyboard, character, and clipboard events.
scrollable / dismissableOwn wheel input and outside-click dismissal without global handlers.
provide / useInject shared services or theme data through the active component context.
ref / expose / slotBridge measured nodes, public component methods, and parent-provided content.
